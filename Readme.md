# Reminder_Bot #

## Add reminder at telegram ! ##

[bot invite link](https://t.me/tkt_reminder_bot)

## set up ##
1. ```pip install -r requirements.txt```
2. ```python main.py```

## how to use the bot ##
1. Use /add_re to add reminder
2. Use /check_re to check reminder
3. Use /remove_re to remove reminder
